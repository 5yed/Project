# Team Project

This repository has been created to store your Team Project.

You may edit it as you like, but please do not remove the default topics or the project members list. These need to stay as currently defined in order for your lecturer to be able to find and mark your work.

# Instructions for running the project

 - 1. Run the backend using `mvn spring-boot:run` in the backend folder.
      Run `mvn clean install` in the backend if there're package issues.
 - 2. Make sure you have `frontend/.env` file with the correct variables set.
      Refer to `frontend/.env.template` for what's needed.
 - 3. Run the frontend using `npm run dev` in the frontend folder.
      Run `npm install` in the frontend if there're package issues.
